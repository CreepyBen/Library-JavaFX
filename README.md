# Library Management System (JavaFX)

A simple yet functional **Library Management System** built with **JavaFX** and **Java 17**.  
This project demonstrates strong understanding of **object-oriented programming**, clean design structure, and a **user-friendly interface** for managing books efficiently.

---

## Features

-  Add new books with title and author  
-  View all added books in a readable format  
-  Clear the current book list  
-  Organized class structure with clear separation of logic and interface  
-  Styled with custom CSS for a clean, modern look  

---

## Tech Stack

- **JavaFX 17** – GUI framework  
- **Java 17** – Core language  
- **CSS** – UI styling  
- **IntelliJ IDEA** – IDE used for development  

---

## How to Run

1. Make sure you have **JDK 17** and **JavaFX SDK 17+** installed.  
2. Clone this repository:
   ```bash
   git clone https://github.com/CreepyBen/Library-JavaFX.git
3. Open the project in IntelliJ IDEA (or your preferred IDE).
4. Set up JavaFX:
      Add the JavaFX SDK lib folder to your module path.
      Under Run Configurations → VM Options, add:
      --module-path "C:\path\to\javafx-sdk-17\lib" --add-modules javafx.controls,javafx.fxml
5. Run LibraryGUI.java as the main class.
