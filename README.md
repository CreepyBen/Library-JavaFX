# Library Management System (JavaFX)

A simple JavaFX-based Library Management System that demonstrates OOP concepts and GUI development in Java.  

## Features
- Add books with title and author
- View all books in the library
- Clean and intuitive graphical interface
- Built using JavaFX for desktop GUI

## Project Structure
- `Book.java` – Represents a book with title and author
- `Library.java` – Handles the collection of books
- `LibraryGUI.java` – Provides the JavaFX user interface

## Technologies Used
- Java 17
- JavaFX SDK
- IntelliJ IDEA

This project was created as a learning exercise to showcase **Java programming, OOP, and GUI skills**.
